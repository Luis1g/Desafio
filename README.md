## **Desafio Practicando Python para Data Science: Challenge Alura Store**

###  **Proposito**

- El propósito del análisis realizado es ayudar al Sr. Juan a decidir que tienda de su cadena Alura Store debe vender para inciar un nuevo emprendimiento. 

### **Estructura**
La estructura del proyecto y organización de los archivos se muestra a continuación;

1. Extracción de los datos: Los datos de las tiendas se encuentran en 4 archivos CSV. Los datos se extraen y organizan en DatasFlames, utilizando la biblioteca pandas.
2. Conocer los datos: Explorar el conjunto de datos.
3. Analisis de facturación: Analisis de los datos de cada tienda: 
    * Ventas por categoria.
    * valoración media de tienda.
    * Productos mas y menos vendidos.
    * Valor de envio promedio por tienda
